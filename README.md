<h3>💻 사용 언어 환경</h3>
  
    1. Java 1.8
    2. Springboot 2.3.12 release

<h3>✨ 맡은 업무 설명</h3>
  <h4> 대략적인 운영자 커뮤니티 게시판 기능은 아래와 같습니다.</h4>
  
     1. 작업 리스트 조회와 검색기능
     2. 작업 등록 (로그인 사용자의 권한에 따라 기능 다름)
       2-1. 작업 승인
         2-1-1. 작업상태 변경
         2-1-2. 승인상태 또는 반려상태
       2-2 작업 완료보고서 작성       
     3. 작업 수정 
     4. 작업 삭제 
     5. 파일 upload
     6. 파일 download
     7. 댓글기능 (+파일 첨부)
